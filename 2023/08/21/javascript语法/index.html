元字符

  • 边界符

    • ^开头 $结尾
      /^哈哈/.test(‘哈哈’) //true
      /^哈哈$/.test(‘哈哈’) //true
      /^哈哈$/.test(‘哈哈哈’) //false

  • 量词

    • *（>=0） +（>=1） ？（0或1）
    • {n} 必须出现n次
    • {n,} >=n
    • {n,m} >=n <=m

  • 字符
    [] 只选1个
    /^[哈]$/.test(‘哈’) //true
    /^[哈]$/.test(‘哈哈哈’) //false
    /^[哈哈]{3}$/.test(‘哈哈哈’) //true
    /[哈哈]/.test(‘哈哈哈’) //true
    /^[a-zA-Z0-9]$/.test(1) //true

    []里加^ 表示取反
    /^[^a-z]$/.test(‘A’) //true

    . 除换行符的任何1个字符

    \d 表示0~9

  • 修饰符
    /规则/修饰符
    i 不区分大小写
    /^aa$/i.test(‘Aa’)//true
    | 或
    /^a|A$/.test(‘aa’)//true
    g 全局
    replace 替换
    console.log(‘2!5!2’.replace(/2/g,1)); //1!5!1

数组常见方法

reduce

数组元素值累加
const arr = [1,2,3]
// p为上一次值， n为当前值(每个元素) [初始值 ]
const b= arr.reduce((p,n)=>p+n)
console.log(b); //6
const b= arr.reduce((p,n)=>p+n , 10)
console.log(b) //16

find

查找（数组）中第一个满足的元素的值
const arr = [{name: ‘大黄’, ages: 2, eyes: 2}]
// item为数组中正在处理的元素
const a = arr.find(item => item.ages === 2)
console.log(a) //{name: ‘大黄’, ages: 2, eyes: 2}
