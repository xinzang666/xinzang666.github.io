数字型

数字型的最大值

console.log(Number.MAX_VALUE);

数字型的最小值

console.log(Number.MIN_VALUE);

无穷大

console.log(Number.MAX_VALUE * 2);

  • Infinity 无穷大

无穷小

console.log(-Number.MAX_VALUE * 2);

  • -Infinity 无穷小

非数字

console.log(‘pink’ - 100)

  • NaN
    • 是否非数字 isNaN
      console.log(isNaN(“pink”));
    • 非数字 true

检测变量数据类型 typeof

console.log(typeof num);

数据类型转换

转为字符型

  • console.log(num.toString());
  • console.log(String(num));
  • console.log(num + “”);

转为数字型

  1. parseInt() 字符型的数字转为数字型 取整
    console.log(parseInt(变量));
  • console.log(parseInt(‘123.8’));
    结果：123
  • console.log(parseInt(‘remsd123’));
    结果：NaN
  1. parseFloat() 字符型的数字转为数字型 可得小数 浮点数
  2. Number()
    console.log(Number(“99.9”));
  3. 隐式转换 - * /
    console.log(‘11’ - 0);
    console.log(‘11’ * 1);
    console.log(‘11’ - ‘123’);

Boolean() 转为布尔型

  • 结果：false
    • console.log(Boolean(‘’));
    • console.log(Boolean(0));
    • console.log(Boolean(NaN));
    • console.log(Boolean(null));
    • console.log(Boolean(undefined));

arguments

  • 存储所有传来的实参
  • 有 length 属性 、 索引
  • function fn(){
    console.log(arguments);
    console.log(arguments.length);
    console.log(arguments[0]);
    for (var i = 0; i < arguments.length; i++){
    console.log(arguments[i]);
    }
    }
    fn(1,2,3);

数组

  1. 定义
  • 字面量
    let names = [1,’i’,true];
  1. 添加
  • 在数组末尾添加
    names.push(‘2’,’o’);
    • 返回数组长度
      console.log(names.push(“jj”)); // 6
  • 在数组开头添加
    names.unshift(‘kk’);
  1. 删除
  • 删除最后(一个)
    names.pop();
    • console.log(names.pop()) //返回删除的元素
  • 删除第(一个)
    names.shift();
  • 删除指定元素(多个)
    names.splice(起始位置，删除几个);
    names.splice(1) //从索引 1 删除到最后

sort

  • 升序
    arr.sort(function(a,b){
    return a - b;
    })
  • 降序
    arr.sort(function(a,b){
    return b - a;
    })

函数

  • 匿名函数
    let fn = function(x, y){} ; fn();
    • 立即执行函数
      (function() {})();
      (function(x,y) { console.log(x + y)} (1,2) );
  • 具名函数
    function fun(){}; fun();

对象

  • 定义
    let obj = {};
    let obj1 = {
    ‘name-first’: ‘1’,
    id: ‘2’
    };

  • 使用
    对象名.属性
    对象名[属性名]
    console.log(obj.name);
    obj[‘name-first’]

  • 修改
    对象名.属性 = 新值

  • 增加
    对象名.（新）属性 = 值

  • 对象的方法
    let obj = {
    score : function(){
    console.log(11)
    }
    };
    obj.score()

  • 遍历对象

    • for in 不推荐遍历数组
      for (let k in obj){
      console.log(k) //属性名 ‘字符串’
      console.log(obj[k]) //值
      }

内置对象

Math

  • Math.abs() 绝对值
  • Math.floor() 向下取整
  • Math.ceil() 向上取整
  • Math.round() 四舍五入
  • Math.random() 随机小数 >=0 <1
  • Math.floor(Math.random() * (10+1)) 随机 0-10 整数
  • 取 N ~ M 的随机整数
    function getRandom(N, M){
    return Math.floor(Math.random() * (M - N +1) + N)
    }

Date

  • var date = new Date();
  • var date = new Date(2024, 10, 1);
    • 返回 11 月，不是 10 月
  • var date = new Date(‘2024-10-1 8:7:8’);
  • date.getFullYear() 年
  • date.getMonth() 月 (0,11)
  • date.getDate() 日
  • date.getDay() 星期几，周日为 0
  • date.getHours() 时
  • date.getMinutes() 分
  • date.getSecond() 秒
  • 距离 1970.1.1 总的毫秒数(时间戳)
    • date.valueOf()
    • date.getTime()
    • var date = +new Date(); +new Date()
    • console.log(Date.now());

DOM

  1. document.querySelector() 只取第一个

  • document.querySelector(‘div’)

  • document.querySelector(‘.box’)

  • document.querySelector(‘#nav’)

  • document.querySelector(‘ul li:first-child’)

  • document.getElementById(‘nav’)

  • document.getElementByTagName(‘div’)

  • document.getElementByClassName(‘box’)

  • 属性选择
    document.querySelector(‘[data-name=text]’)

  1. 获取元素内容 innerText
  • const box = document.querySelector(‘.box’)
    console.log(box.innerText)
    box.innerText = ‘<strong>box</strong>‘ // 效果：不加粗 只修改内容，不解析标签
  1. 解析标签 innerHTML
  • box.innerHTML = ‘<strong>box</strong>‘ // 效果：加粗

操作类名

  • className (覆盖)
    <style>.box{} .nav{}</style>
    <div></div>
    <script>
    const box= document.querySelector(‘.box’)
    box.className = ‘box nav’
    </script>

  • 追加类 add()
    box.classList.add(‘active’)

  • 删除类 remove()
    box.classList.remove(‘active’)

  • 切换类 toggle() 有就删除，没有追加
    box.classList.toggle(‘active’)

自定义属性

<div data-id="1" data-spm="2">1</div>
const one = document.querySelector(‘div’)
console.log(one.dataset.id) //one.dataset 得到一个对象
console.log(one.dataset.spm) //得到 2

定时器-间歇函数

  • setInterval(函数, 间隔时间)
  1. 匿名函数
    setInterval(function() {console.log(1)} , 1000)
  2. 函数
    function fn(){console.log(1)}
    setInterval(fn , 1000) //函数名不加小括号
  • 关闭定时器
    let t = setInterval(function(){console.log(1)}, 1000)
    clearInterval(t) //t 为常量

事件类型

  1. 点击
    addEventListener(‘click’, function(){});
  2. 鼠标经过
    addEventListener(‘mouseenter’,function(){}) //无冒泡
    addEventListener(‘mouseover’,function(){}) //有冒泡
  3. 鼠标离开
    addEventListener(‘mouseleave’, function(){}) //无冒泡
    addEventListener(‘mouseout’,function(){}) //有冒泡
  4. 焦点
    addEventListener(‘focus’,function(){})
  5. 失去焦点
    addEventListener(‘blur’,function(){})
  6. 键盘事件（按下键盘触发）
    addEventListener(‘keydown’,function(){})
    addEventListener(‘keyup’,function(){})
  7. 输入文本
    addEventListener(‘input’,function(){})

事件对象

阻止冒泡

e.stopPropagation() //e 为事件对象

环境对象 this

函数运行的环境

  • window
  • 谁用，this 指向谁

事件移除

  1. L0-空对象
    btn.onclick = function(){}
    btn.onclick = null
  2. L2-匿名函数无法移除
    function fn(){}
    btn.addEventListener(“click”,fn)
    btn.removeEventListener(“click”,fn)

tab 切换案例
const ul = document.querySelector(‘.tab-nav ul’)
ul.addEventListener(‘click’,function(e){
document.querySelector(‘.tab-nav .active’).classList.remove(‘active’)
e.target.classList.add(‘active’)
const i = +e.target.dataset.id
document.querySelector(‘.tab-content .active’).classList.remove(‘active’)
document.querySelector(.tab-content .item:nth-clild(${i + 1})).classList.add(‘active’)
})

页面尺寸事件

  • 浏览器窗口大小发生变化时触发
    window.addEventListener(‘resize’,function(){
    console.log(1)
    })
  1. 获取元素宽高

  • 不包含 border，margin 只读
    div.clientWidth
    div.clientHeight

  • 内容+padding+border 只读
    如果盒子隐藏，获取结果为 0
    offsetWidth,offsetHeight

  1. 获取位置
  • 只读
    获取元素距离自己带定位的父级元素的左、上距离
    offsetLeft,offsetTop

日期对象

  1. 年月日

  • 实例化
    const date = new Date()
    const date = new Date(‘2022-10-1 08:30:00’)

  • 年 date.getFullYear()

  • 月 date.getMonth() + 1 //0~11

  • 日 date.getDate()

  • 星期 date.getDay() //0~6

  • 时 date.getHours()

  • 分 date.getMinutes()

  • 秒 date.getSeconds()

  • date.toLocaleString() //2022/10/1 08:30:00

  • date.toLocaleDateString() //2022/10/1

  • date.toLocaleTimeString() //08:30:00

  1. 时间戳 -毫秒数
  • 法 1，必须实例化
    const date = new Date()
    date.getTime()
  • 法 2 +new Date()
  • 法 3 Date.now()
