选择器

多类名选择器

css：.box{}（样式.定义）

  • 各个类名用空格隔开
    html 例：<div class="box color"></div>

id 选择器

样式#定义
css：#box{}
html 例：<div id="box"></div>

通配符选择器

css 例：*{}

标签选择器

css 例： div{}

字体

  • 复合属性：font: font-style font-weight font-size(必须有)/line-height font-family(必须有);
  1. 例：微软雅黑字体
    font-family: “微软雅黑”;
  2. 字体大小 （默认 16px）
    font-size: 1px;
  3. 字体粗细
  • 加粗 - 注：700 等同 bold
    font-weight: bold;
    font-weight: 700;
  • 变细 - 注：400 等同 normal
    font-weight: normal;
    font-weight: 400;
  1. 文字样式
  • 倾斜
    font-style: italic;
  • 正常
    font-style: normal;
  • 注：可将 em,i 标签变不倾斜

文本

  1. 颜色
    color: pink;
    color: #ccc;
    color: rgb();
  2. 水平对齐
    text-align: center;
  • left 左对齐 right 右对齐
    • 图片居中：给图片父级加属性
  1. 装饰
  • 没有 (取消下划线)
    text-decoration: none;
  • 下划线
    text-decoration: underline;
  • 删除线
    text-decoration: line-through;
  • 上划线
    text-decoration: overline;
  1. 文本缩进 (可取负值)
    text-indent: 20px;
  • 缩进两个字
    text-indent: 2em;
  1. 行间距 (行高)
    line-height: 16px;

复合选择器

  1. 后代选择器
    父 子 (孙)
    ul li{}
  2. 子元素选择器
    父 子
    ul > li{}
  3. 并集选择器
    div,
    p {}
  4. 伪类选择器
    1. 链接伪类(顺序：lvha)
    • a:link{} : 未访问的链接
    • a:visited{} :访问过的链接
    • a:hover : 鼠标经过链接
    • a:active : 按下链接
    1. focus 伪类
    • 获得光标的表单元素
      例：input:focus{}

显示模式

  1. 块元素
  • h1~h6、p、div、ul、ol、li
    <h1>``<p>元素不能放块级元素
  1. 行内元素
  • a、strong、b、em、i、del、s、ins、u、span
  • 宽高直接设置无效
  • 行内元素里只容纳行内元素
    <a>里可放块级元素
  1. 行内块元素
  • img、input、td
  • 可设宽高
  1. 元素显示模式转换
  • 块
    display: block;
  • 行内
    display: inline;
  • 行内块
    display: inline-block;

背景

复合写法（无规定）：background: 颜色 图片 平铺 滚动 位置

  1. 背景颜色
    透明：background-color: transparent;
  2. 背景图片
    background-image: url();
  3. 背景平铺
    • 不平铺：
      background-repeat: no-repeat;
    • 平铺：
      background-repeat: repeat;
    • 横向平铺：
      background-repeat: repeat-x;
    • 纵向平铺：
      background-repeat: repeat-y;
  4. 图片位置
    水平：left center right
    垂直：top center bottom
    1. background-position: 水平 垂直;
      只写一个值时，另一个值默认为 center
    2. 精确单位：background-position: 10px 10px;
    • 只写一个值时，默认为 x 坐标
    1. 混合单位：background-position: 10px center;
  5. 背景固定
  • 固定：
    background-attachment: fixed;
  • 滚动（默认）
    background-attachment: scroll;
  1. 背景色半透明
  • 最后一个值区间为 0-1
    background: rgba(0,0,0,0.3);

盒子边框

  • 复合写法（没规定）：border: 粗细 样式 颜色;
    border-top: 粗细 样式 颜色;
  1. 边框粗细
    border-width: 1px;
  2. 边框样式
    • 实线
      border-style: solid;
    • 虚线
      border-style: dashed;
    • 点线
      border-style: dotted;
  3. 边框颜色
    border-color: pink;

表格细线边框

  • 相邻边框合并：
    border-collapse: collapse;
